prime_t prime_new ( )

make new prime generation instance.

generates new instance of the prime_t struct, which is used for caching generated primes and needs to be passed to all prime functions.

Returns
a prime_t instance.

Examples

// make new primes instance
prime_t primes = prime_new();
// generate a prime
assert(prime_nth(&primes, 3) == 7);
// free prime instance
prime_free(&primes);
