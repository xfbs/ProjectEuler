◆ RUN

#define RUN (   t)
Value:
printf(" - %s: ", #t); \

run a single test function with the provided name

this function will run the given function and prime some helpful messages to stdout.

this function is meant to be called in a TESTS() block.

Parameters
tfunction to test, must be a void function.

Examples

void test_something() {
assert(true);
}
TESTS() {
RUN(test_something);
}

◆ TESTS

#define TESTS ( )
Value:

define the main testing function.

this function may only be called once, and with it you define a function that will run all the tests that you specify.

inside the function, call RUN() to run the tests you want to run.

Examples

void test_something() {
assert(5 == 5);
assert(25 == (5 * 5));
}
TESTS() {
RUN(test_something);
}
