Detailed Description

Generic functions to operate on various vector types implemented as macros.

The macros in this file serve as generic functions that can be used in place of the specialized functions of each vector type.

For example, instead of using vec32_len(), you can use vec_len(), and the macro will automatically select the right function to use.

See also
vecp, vec8, vec16, vec32, vec64

Examples

reserves some space in vector to add large amounts of data.

// allocates new vec32
vec32_t vec = vec32_new(0, 0);
// reserve enough space
size_t data_size = 1000000;
vec_reserve(&vec, data_size);
// add data
for(size_t cur = 0; cur < data_size; cur++) {
vec_push(&vec, 9);
}
vec_free(&vec);
