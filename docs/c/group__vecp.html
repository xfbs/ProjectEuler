a data type and functions for handling pointer vectors

◆ vecp_compare

typedef int(* vecp_compare) (const void *, const void *)

comparison function between two elements.

functions that have this type can be used as sorting functions with vecp_sort(). functions are expected to behave in the same way as comparison functions for the qsort() function of the standard library.

◆ vecp_alloc()

vecp_t* vecp_alloc ( size_t  len,
void *  fill 
)

like vecp_new(), but allocates with malloc, used to get pointer.

Parameters
leninitial length ov the vecp_t.
fillwhat it should be pre-filled with.
Returns
a pointer to a malloc-allocated vecp_t.
Warning
unlike vecp_new(), which just returns a vecp_t that you can keep on the stack, this method returns a pointer to malloc'ed memory, which means that in addition to calling vecp_free() to free the internals, you need to call free() on the pointer itself after you're done using it.

Examples

creating a new empty pointer vector

// creates an empty vecp
vecp_t *vec = vecp_alloc(0, NULL);
vecp_free(vec);
free(vec);

creating a pre-filled pointer vector

// element to point to
int a;
// creates a pre-filled vecp
vecp_t *vec = vecp_alloc(10, &a);
assert(vecp_len(vec) == 10);
assert(vecp_get(vec, 0) == &a);
assert(vecp_get(vec, 9) == &a);
vecp_free(vec);
free(vec);

internally, this resets the vector's size to 0, but it does not shrink the array used to hold the data. if you want to shrink the container, you can call vecp_free() on it and make a new vector with vecp_new().

◆ vecp_free()

void vecp_free ( vecp_t *  v)

free()'s the data held inside the pointer vector.

Parameters
vthe vector that will have its data released.
Warning
this function only releases the data held inside a vecp_t, not the vecp_t itself. if the vecp_t has been produced by vecp_alloc(), free() must also be called on it to release the structure itself.

Examples

freeing a vector on the stack

// allocate vector on the stack
vecp_t vec = vecp_new(0, NULL);
// free vector contents
vecp_free(&vec);

freeing a vector on the heap

// allocate new vector on the heap
vecp_t *vec = vecp_alloc(0, NULL);
// free vector contents
vecp_free(vec);
// free vector itself
free(vec);

Returns

Warning

Examples

get elements from a pre-filled vector

vecp_free(&vec);

// data to point to
int a, b;
// creates a new empty vecp
vecp_t vec = vecp(0, NULL);
// push some data into it
vecp_push(&vec, &a);
vecp_push(&vec, &b);
// check values
assert(vecp_get(&vec, 0) == &a);
assert(vecp_get(&vec, 1) == &b);
vecp_free(&vec);

◆ vecp_index()

size_t vecp_index ( const vecp_t *  v,
void *  ptr 
)

perform reverse lookup on vector

Parameters
vvector to perform lookup in
ptrpointer to look up
Returns
index of ptr in v, or SIZE_MAX if data does not exist in v.

for a given vector v and element data, searches the vector for occurences of data, returning the index of the first occurence or SIZE_MAX to indicate that the element does not exist.

Examples

// allocates new vector
vecp_t vec = vecp_new(0, NULL);
// data to point to
int a, b, c, d;
// adds some data to vec
vecp_push(&vec, &a);
vecp_push(&vec, &b);
vecp_push(&vec, &c);
// find index of some elements in vector
assert(vecp_index(&vec, &a) == 0);
assert(vecp_index(&vec, &b) == 1);
assert(vecp_index(&vec, &c) == 2);
assert(vecp_index(&vec, &d) == SIZE_MAX);

◆ vecp_reserve()

void vecp_reserve ( vecp_t *  v,
size_t  size 
)

reserves the given amount of space for adding new elements.

Parameters
vthe vector to reserve space for
sizehow much additional space to reserve

when adding large amount of pointers to a vector, and when the amount is known in advance, this method can be used to reserve space to prevent multiple reallocations of the data while adding them.

Examples

reserves some space in vector to add large amounts of data.

vecp_t vec = vecp_new(0, NULL);
// reserve enough space
size_t data_size = 1000000;
vecp_reserve(&vec, data_size);
// add data
for(size_t cur = 0; cur < data_size; cur++) {
vecp_push(&vec, NULL);
}
vecp_free(&vec);

vecp_push(&vec, NULL);
// set new values
vecp_set(&vec, 0, &a);
vecp_set(&vec, 1, &b);
// check values
assert(vecp_set(&vec, 0) == &a);
assert(vecp_get(&vec, 1) == &b);
vecp_free(&vec);
